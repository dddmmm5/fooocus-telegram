```bash
git clone https://github.com/your-github-username/your-repo-name.git
cd your-repo-name

```

## Usage
Set your Telegram Bot Token and Fooocus API Key in .env file:  
```bash
BOT_TOKEN=123123123:wsdfasdf:12341234
FOOOCUS_IP=127.0.0.1
FOOOCUS_PORT=8080
```

## Run the bot:
```bash
python main.py
```
