## Usage
Set your Telegram Bot Token and Fooocus API Key in .env file:  
```bash
FOOOCUS_IP = 0.0.0.0
FOOOCUS_PORT = 8080
BOT_TOKEN = YOUR_BOT_TOKEN
ADMIN_ID = 123123123123
GROUP_ID = -123123123123123
# Database
PG_DB = postgres
PG_PASS = 1234567890
PG_USER = postgres
PG_HOST = localhost 
```

## Run the bot:
```bash
python main.py
```
